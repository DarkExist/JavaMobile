#Assembly instructions

1. Clone the repo
  ```bash
  git clone https://github.com/DarkExist/JavaMobile
  ```
1. Move to the repo
  ```bash
  cd JavaMobile
  ``` 
3. Start assembly
  ```bash
  ./gradlew assembleRelease
  ``` 
4. The final file is stored in the folder
  ```bash
  cd app\build\outputs\apk\release
  ```
